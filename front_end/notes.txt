  07 - Useful String Methods
        - trim() remove spaces from given starting
        - length - get lenght of given string
        - toUpperCase() - String converted into UPPER case 
        - toLowerCase() - String converted into lower case
        - slice(start Index,end index) - String Start Index and End Index
        - .split() method - split string into array
        - .join() method - join array into string
        - .includes() method - check if string contains given value
        - .indexOf() method - get index of given value in string
        - .charAt() method - get character at given index in string
        - .substring() method - get substring from given index to end of string
        - .replace() method - replace given value with new value in string
        - .search() method - search given value in string and return index of first occurrence
        - .match() method - search given value in string and return array of all occurrences

  13 - if else, truthy and falsy values
        - if else - used to execute block of code based on condition

        - condition are false it execute else block of code
        - truthy values - values that are true in if condition
        - falsy values - values that are false in if condition

  14 - Ternery operator
        - ternery operator - used to execute block of code based on condition
        - eg. condition ? value if true : value if false

  15 - nested if else
        - nested if else - used to execute block of code based on multiple conditions
        - eg. if (condition1) {
                  if (condition2) {

               }
               
   16 - if elseif

        - eg. if (condition1) {
